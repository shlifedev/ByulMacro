# DS AUtomation
Very Simple Customizingable Macro
(외주 프로젝트)



# Features
- GUI
- Command Group Manager
- Keyboard & Mouse Event Hook and Record
- OpenCV Image Matcher
- Support Multiple Monitor 
- Windows Message Sender For IO
- Windows Message Hooker
